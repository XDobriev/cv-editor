# Резюме-конструктор — Хамзат Добриев · Frontend Developer

Личный конструктор резюме: правишь содержимое под вакансию в визуальном редакторе,
получаешь онлайн-версию и скачиваешь настоящий текстовый PDF. Плюс лендинг-портфолио.
Эстетика «Dev / Techno» (IBM Plex Sans + Mono, тёплая бумага, терракотовый акцент).
Чистая статика без сборки.

## 🔗 Живая версия

- **Портфолио (главная):** https://xdobriev.github.io/cv-editor/
- **Онлайн-резюме → PDF:** https://xdobriev.github.io/cv-editor/resume.html
- **Редактор:** https://xdobriev.github.io/cv-editor/editor.html

## Архитектура

Единый источник данных и единый рендерер — никаких копий контента:

```
resume-data.js    →  window.RESUME_DATA  (контент)  +  window.PORTFOLIO (доп. для лендинга)
resume-render.js  →  renderResumeHTML(data) + хелперы (esc, inlineRich, buildHref, …)
        │
        ├─ editor.html   форма + живое превью (через renderResumeHTML), автосохранение, PDF
        ├─ resume.html   онлайн-резюме A4 + экспорт в PDF (тот же рендерер; подхватывает
        │                личные правки из localStorage редактора, если они есть)
        └─ index.html    лендинг-портфолио (cv.js → WebCV) из тех же данных
```

## Состав

| Файл | Назначение |
|------|------------|
| `resume-data.js`   | **Единый источник** контента резюме + данные портфолио |
| `resume-render.js` | Общий рендерер A4-резюме и текстовые хелперы |
| `editor.html`      | Визуальный редактор (правка, темизация, экспорт/импорт JSON, печать) |
| `resume.html`      | Онлайн-резюме формата A4 с экспортом в текстовый PDF |
| `index.html` + `cv.js` | Лендинг-портфолио (hero · проекты · опыт · стек · контакты) |
| `drafts/`          | Архив: альтернативные вёрстки резюме и старая печатная версия |

## Как пользоваться

1. Открой **редактор** (`editor.html`), поправь резюме под вакансию — всё сохраняется автоматически.
2. Кнопка **«Скачать PDF»** → в диалоге печати выбери «Сохранить как PDF» (текст останется выделяемым).
3. Для стабильной публичной версии: **«Экспорт»** JSON → положи данные в `resume-data.js` → закоммить.
